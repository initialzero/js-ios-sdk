Inherits from JSRESTBase : NSObject
Declared in JSRESTResource.h
JSRESTResource.m

Overview

Provides wrapper methods for JSRESTBase to interact with the repository JasperReports server REST API. This object puts at disposal a set of methods for getting list of resources, getting single resource (also getting with query data), creating, modifying and deleting resource (all sending methods are asynchronous and use base sendRequest method from JSRESTBase class).

Contains two types of methods which differs by last parameter: first type uses delegate and puts request result into it, second type uses pre-configuration block for setting additional parameters of JSRequest object. Pre-configuration block implicitly supports finishedBlock usage instead delegate object (or use them both), setting custom timeoutInterval, custom query parameters, requestBackgroundPolicy etc.

@author Vlad Zavadskii vzavadskii@jaspersoft.com @author Alexey Gubarev agubarev@jaspersoft.com
