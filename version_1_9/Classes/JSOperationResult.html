Overview

Uses to hold the different results of a REST call to JasperReports server. Result can be a list of object (single object represents as list with one object) or binary data (downloaded file)

@author Giulio Toffoli giulio@jaspersoft.com @author Vlad Zavadskii vzavadskii@jaspersoft.com
